Message Bar Demo | <uif-message-bar>

In order for this demo to work you must first build the library in debug mode.

Default

To render message banner, use the following markup:
<uif-message-bar> <uif-content>lorem ipsum dolor sit amet, a elit sem interdum consectetur adipiscing elit</uif-content> <uif-link ng-href="http://ngofficeuifabric.com">Link text</uif-link> </uif-message-bar>

lorem ipsum dolor sit amet, a elit sem interdum consectetur adipiscing elit Link text

Example with uif-type: error

<uif-message-bar uif-type="error"> <uif-content>lorem ipsum dolor sit amet, a elit sem interdum consectetur adipiscing elit</uif-content> <uif-link ng-href="http://ngofficeuifabric.com">Link text</uif-link> </uif-message-bar>
